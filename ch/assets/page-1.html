Three.js

Three.js 是一套基於 WebGL 開發出的 Javascript 函式庫，
它提供了比 WebGL 更簡單的 Javascript API，讓開發者能夠輕易在瀏覽器做 3D 繪圖。

Three.js比較偏向3D類型的canvas套件

官網Demo範例

優點:

可以製作複雜模型特效，支援度跨平台良好

缺點:

技術門檻較高，文件類型種類最多，開發時程較長
